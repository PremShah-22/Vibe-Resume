# 🚀 Vibe Resume

An AI-powered resume builder that helps users create beautiful, professional resumes in minutes. Simply enter your details, customize the design, and export a polished resume ready for job applications.

## ✨ Features

* 🔐 Secure User Authentication
* 📝 Easy Resume Creation & Editing
* 🎨 Modern Resume Templates
* 📄 PDF Export Functionality
* 💾 Cloud Storage for Resume Data
* 📱 Fully Responsive Design
* ⚡ Fast and Intuitive User Experience
* 🌙 Clean Modern UI

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router

### Backend & Database

* Firebase Authentication
* Cloud Firestore

### Additional Libraries

* Axios
* React Icons
* React Hook Form
* html2pdf.js (for PDF generation)

## 🚀 Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm or yarn
* Firebase Project

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/vibe-resume.git
```

2. Navigate to the project directory

```bash
cd vibe-resume
```

3. Install dependencies

```bash
npm install
```

4. Configure Firebase

Create a `.env` file in the root directory and add:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

5. Start the development server

```bash
npm run dev
```
